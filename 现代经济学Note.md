# 现代经济学 笔记
## 一、经济学的最底层逻辑
### 1. 什么是经济学？
经济学本质上是**有限资源条件下的人类协作学**。
### 2. 稀缺性与资源配置
经济学存在的前提就是**资源有限**。

资源永远是有限的。
人的需求几乎无限。
经济学研究的是**有限资源如何配置**。
### 3. 机会成本
机会成本是指为了得到某样东西，而**放弃的所有其他选择中，价值最高的那个**。

人们在做出决策时，应考虑到付出的成本不仅包括决定要做的那件事的成本，也应考虑到机会成本。

### 4. 分工为什么能极大提高生产效率？
分工可以显著**提高生产效率**。

分工能提高效率的原因：**熟能生巧**、**可减少切换成本**、**促进技术创新**。

分工越细，通常**需要更大的市场**来支撑。

分工不是无限细越好，因为**协调和管理也有成本**。

### 5. 供给与需求
价格不是简单由成本决定，而是由**供给和需求**共同决定。

需求增加，价格通常上涨。
供给增加，价格通常下降。
供给减少，价格通常上涨。
需求减少，价格通常下降。

### 6. 价格上涨 ≠ 需求增加
价格本身不能告诉我们是哪一边发生了变化。
所以经济学分析市场时，首先要判断：是**需求**变化，还是**供给**变化。

注意区分**需求**与**需求量**：

    需求量变化：价格变化导致购买数量变化。
    需求变化：价格不变，消费者偏好、收入、人口等因素变化，导致愿意购买更多或更少。
